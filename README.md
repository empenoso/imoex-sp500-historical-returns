# Историческая доходность рынков: IMOEX vs S&P 500 📈

Этот репозиторий содержит Python-скрипт для анализа результатов стратегии "Buy & Hold" (купи и держи) на российском (IMOEX) и американском (S&P 500) фондовых рынках.

Главная цель публикации этого кода — **прозрачность**. Каждый желающий может скачать исходные данные, запустить скрипт на своём компьютере и лично верифицировать все расчёты, графики и вероятности просадок.

## Откуда данные?

Я взял данные американского рынка отсюда: https://github.com/datasets/s-and-p-500

Для российского рынка: https://www.moex.com/ru/factsheet/history

Для России хотел взять Индекс полной доходности МосБиржи (MCFTR), но он только с 26.02.2003 доступен, а Индекс МосБиржи (IMOEX) с 22.09.1997.

## Что делает скрипт?
1. **Считает реальную доходность** (Total Return) и среднегодовую доходность (CAGR) для горизонтов в 1, 3, 5, 10, 15 и 20 лет.
2. **Анализирует кризисы:** показывает, что происходило с капиталом при инвестировании за 1–3 года до и после крупных обвалов (1929, 2008, 2020, 2022 и др.).
3. **Генерирует лог для LLM:** выводит подробную текстовую статистику в терминал, которую удобно скопировать в ChatGPT / Claude для дальнейшего текстового анализа.
4. **Строит графики в высоком разрешении (300 DPI):**
   * Боксплоты распределения доходностей.
   * Таблицы доходности по десятилетиям.
   * Графики вероятности получения убытка в зависимости от срока удержания.
   * **Тепловые карты (Heatmaps)** — визуализация доходности для каждого месяца входа в рынок за всю историю наблюдений. 

## 🚀 Как запустить самому

**1. Клонируйте репозиторий:**

```bash
git clone https://github.com/empenoso/imoex-sp500-historical-returns.git
cd imoex-sp500-historical-returns
```

**2. Установите зависимости:**

Для работы потребуются стандартные библиотеки анализа данных:
```bash
pip install pandas numpy matplotlib
```

**3. Подготовьте данные:**

В корневой папке скрипт ожидает найти два CSV-файла:
* `s-and-p-500.csv` — исторические данные S&P 500 (Ожидаемые столбцы: `Date`, `SP500`).
* `IMOEX.csv` — исторические данные Индекса Мосбиржи (Ожидаемые столбцы: `TRADEDATE`, `CLOSE`, разделитель `;`).

*(Примечание: вы можете скачать актуальные данные с Yahoo Finance или сайта Мосбиржи).*

**4. Запустите расчёт:**

```bash
python graphs.py
```
